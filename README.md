# Projeto_Anime_v2.0 #

## Sobre

O projeto **Anime v2.0** é uma aplicação que foi criada dentro da disciplina de C206 e C207 (Programação orientada a objetos e Banco de dados) com o intuito de colocarmos em prática o conteúdo estudado durante o curso em uma aplicação que utiliza um banco de dados relacional SQL, interagindo com a plataforma desenvolvida em Java.

---

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias

- [Java](https://www.java.com/pt-BR/about/whatis_java.jsp)
- [SQL](https://www.alura.com.br/artigos/o-que-e-sql#:~:text=SQL%20significa%20Standard%20Query%20Language,SQL%20Server%2C%20entre%20muitos%20outros.)
